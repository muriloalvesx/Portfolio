Minhas Habilidades

HTML

HTML (HyperText Markup Language) é a linguagem padrão para estruturar e organizar conteúdo na web. Utilizando tags, define elementos como cabeçalhos, parágrafos e links, sendo essencial para criar a estrutura básica de páginas na interne

CSS

CSS (Cascading Style Sheets) é a linguagem que dá estilo e formato a elementos HTML em páginas web, permitindo definir cores, fontes e layout para uma apresentação visual atrativa e consistente.

JavaScript

JavaScript é uma linguagem de programação para web que torna as páginas interativas. Ela é executada no navegador, permitindo manipulação dinâmica de elementos, validação de formulários e atualizações de conteúdo sem recarregar a página.
